Lucidity

Novemeber 2013 · www.nodenexus.com:3000

Lucidity began as an excuse to make a WebSockets application for an introductory web development class, which I was teaching at the time. I wanted to write an application which the students could use throughout the class before ultimately making it themselves. The original backend was a mere twelve lines of code, but it quickly grew to implement many popular features of chat clients such as IRC and Twitch chat. Lucidity can be embedded in other applications and allows for customization by both clients and administrators. Check it out below!
